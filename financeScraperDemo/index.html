Finance Scraper

App written in Kotlin and python to scrape financial data from pdf documents and visualise through graphs and databases. Using chaquo library for running python within the Kotlin framework (better versitility for PDF parsing & data manipulation), SQLite file for CRUD operations on the data (better control and processing speed than ROOM) and vico for charting of data.

Features

  • Scrape date, amount, balance and description from transaction data
  • Visualise transaction history with time-dependent controls

Supported sources

  • Santander (debit)
  • Chase (debit, saver, round-up)
  • AJ Bell (LISA)
  • Trading212
  • eToro

Technologies Used

  • Kotlin for app development
  • Python for data processing and scraping
  • SQLite for database storage
  • Vico for data visualization
